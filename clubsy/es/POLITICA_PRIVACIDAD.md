## 2. Datos que podemos tratar
Según el uso de la app, podemos tratar:
- Datos de cuenta: nombre, correo, identificadores de usuario.
- Datos de perfil: avatar/foto, idioma, preferencias.
- Datos operativos: clubes, equipos, eventos, asistencia, invitaciones.
- Datos técnicos: dispositivo, registros de error, métricas técnicas.
- Datos de ubicación (si aplica y con permisos): para funciones geolocalizadas.

## 3. Finalidades del tratamiento
- Prestar y mantener el servicio.
- Gestionar autenticación, seguridad y prevención de fraude.
- Permitir funciones solicitadas por el usuario/club.
- Mejorar rendimiento y experiencia del producto.
- Cumplir obligaciones legales.

## 4. Bases legales (si aplica GDPR/LOPDGDD)
Podemos basarnos en:
- Ejecución del contrato/servicio.
- Consentimiento del usuario (cuando sea requerido).
- Interés legítimo (seguridad, mejora y operación).
- Cumplimiento de obligación legal.

## 5. Conservación de datos
Conservamos los datos durante el tiempo necesario para cumplir las finalidades indicadas y obligaciones legales, o hasta solicitud válida de supresión cuando proceda.

## 6. Cesiones y encargados
